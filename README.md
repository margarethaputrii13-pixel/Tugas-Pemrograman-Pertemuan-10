1. mulai Program dijalankan oleh pengguna.

2. Inisialisasi Dictionary

Program membuat dictionary kosong:
	•	data_mahasiswa = {}
Dictionary ini digunakan untuk menyimpan data mahasiswa dengan:
	•	Key   : NIM
	•	Value : Nama dan Nilai mahasiswa

3. Tampilkan Menu & Input Pilihan

Program menampilkan menu pilihan:
	1.	Tambah Data
	2.	Ubah Data
	3.	Hapus Data
	4.	Tampilkan Data
	5.	Cari Data
	6.	Keluar

Pengguna memasukkan nomor menu yang diinginkan.

4. Percabangan Berdasarkan Pilihan

Pilihan 1 – Tambah Data
	•	Pengguna memasukkan NIM, Nama, dan Nilai
	•	Data disimpan ke dalam dictionary
	•	Program kembali ke menu utama

Pilihan 2 – Ubah Data
	•	Pengguna memasukkan NIM
	•	Program mengecek apakah NIM ada
	•	Jika ada → data nama/nilai diperbarui
	•	Jika tidak → tampil pesan data tidak ditemukan
	•	Kembali ke menu

Pilihan 3 – Hapus Data
	•	Pengguna memasukkan NIM
	•	Program mengecek keberadaan NIM
	•	Jika ada → data dihapus
	•	Jika tidak → tampil pesan error
	•	Kembali ke menu

Pilihan 4 – Tampilkan Data
	•	Program menampilkan seluruh data mahasiswa yang tersimpan
	•	Kembali ke menu

Pilihan 5 – Cari Data
	•	Pengguna memasukkan NIM
	•	Program menampilkan data mahasiswa jika ditemukan
	•	Kembali ke menu

Pilihan 6 – Keluar
	•	Program dihentikan

5. Selesai

Program berakhir ketika pengguna memilih menu keluar.
